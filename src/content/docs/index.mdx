## What this is

A single place where I collect what I'm learning in tech:

- **Topics** — evergreen, categorized knowledge base entries. Things like "RAG", "Kubernetes networking", "Postgres indexes".
- **Posts** — dated write-ups. What I learned this week, project notes, experiments.

Content is authored in plain markdown so the same source renders nicely on GitHub, here as a site, and can be pasted into dev.to / Medium / LinkedIn.

See the [repo on GitHub](https://github.com/waggertron/tech-learning).
